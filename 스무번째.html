        function test(){

            for(let i=0; i<=svalue; i++){
                //console.log(i)
                //return i

            }

        }
  • 해설
  • answer 부분에 문자열을 넣을땐, 빈값 ''을 넣어주었지만, 숫자를 넣을땐 0,1,null을 넣어준다. 지금은 곱하기 이므로 1을 넣어주었다.

요구한 답

        console.log(Math.pow(parseInt(n[0], 10), parseInt(n[1], 10)));
  • 해설
  • Math.pow을 사용

문제2) 몫과 나머지

공백으로 구분하여 두 숫자가 주어집니다.
두번째 숫자로 첫번째 숫자를 나누었을 떄 그 몫과 나머지를 공백으로 구분하여 출력하세요.

        입출력
        입력: 10 2
        출력: 5 0

내가 쓴 답

        function test2(){
            const fnValue = document.getElementById('fn').value;
            const snValue = document.getElementById('sn').value;

            console.log(fnValue,snValue);
            const sAnswer = fnValue/snValue;
            const ssAnswer = fnValue%snValue;
            console.log(sAnswer,ssAnswer)
        }
  • 해설
  • 몫은 나눗셈을 나머지는 나머지 연산자를 이용하여 각각의 변수에 담아주었다.

요구한 답

                ```jsx
        const n = prompt('수를 입력하세요.').split(' ');

        const result = Math.floor(parseInt(n[0], 10) / parseInt(n[1], 10));
        const left = parseInt(n[0], 10) % parseInt(n[1], 10);

        console.log(result, left);
        ```
  • 해설
  • const result는 Math.floor를 써서 소수점 아래는 버리고 나눔
  • const left는 나머지 연산자를 사용
  • 내 코드가 더 낫지않나..
