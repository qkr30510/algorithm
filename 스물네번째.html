문제1)객체 만들기

첫번째 입력에서는 학생의 이름이 공백으로 구분되어 입력되고, 두번째에는 그 학생의 수학 점수가 공백으로 구분되어 주어집니다. 두 개를 합쳐 학생의 이름이 key이고 value가 수학 점수인 객체를 출력해주세요. 

        입력
        Yujin Hyewon
        70 100
        
        출력
        {'Yujin': 70, 'Hyewon': 100}        
    
        const name = prompt('이름을 입력해주세요.').split();
        const math = prompt('점수를 입력해주세요.').split();    
        const aa = {};
        
        for(let i = 0; i < name.length;  i++){
            aa[name[i]] = parseInt(math[i], 10);            
        }
        console.log(aa);
  • 해설
  • split() 메서드를 통해 배열로 만들어준다.
  • 객체[값]이기 때문에 aa[name]은 aa객체의 값name =은 math이다
  • 하나의 규칙이다. aa[1번쨰는 key값 = value값]
