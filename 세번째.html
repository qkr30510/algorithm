문제 이름: 시험 성적

백준 알고리즘
  • 시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 나머지 점수는 F를 출력하는 프로그램을 작성하시오.

풀이

  • else if를 사용함.
  • &&를 이용하여 맨처음에 0 ~ 100의 제한 값을 두었다.
  • 그 이후 부터는 그냥 간단하게 ">=80"처럼 일정한 숫자 이상 이면 실행 되게 넣어주었다.

시도했던 방법

                function check_value() {
                    var inputValue = document.getElementById('check_date').value;
                    console.log(inputValue);
        
                    if (!inputValue) {
                        alert("입력해주세요")
        
                    } else if (101 < inputValue) {
                        alert("100보다 큰 값은 없습니다.")
        
                    } else if (101 < inputValue && inputValue >= 90) {
                        console.log("점수는:a")
        
                    } else if (90 < inputValue && inputValue >= 80) {
                        console.log("점수는:b")
        
                    } else if (79 < inputValue && inputValue >= 70) {
                        console.log("점수는:c")
                    } else if (69 < inputValue && inputValue >= 60) {
                        console.log("점수는:d")
                    } else {
                        console.log("점수는:f")
                        return;
                    }
