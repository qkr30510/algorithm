문제1) for를 이용한 기본 활용

1부터 100까지 모두 더하는 Code를 부분에 완성하세요. for를 사용해야합니다.

        let s = 0;
        //pass
        console.log(s)

답

        let s = 0;
        for(let i=0; i<=100; i++){
            console.log(s+=i)
        }

문제2) 게임 캐릭터 클래스 만들기

다음 소스코드에서 클래스를 작성하여 게임 캐릭터의 능력치와 '파이어볼'이 풀력되게 만드시오. 소스 코드 수정하면 안됨

        데이터
        <여기에 class를 작성하세요>

        const x = new Wizard(545,,210, 10);
        console.log(x.health, x.mana, x.armor);
        x.attack();

        출력
        545 210 10
        파이어볼

답

        
        const Wizard = class Wizard {
            constructor (health, mana, armor) {
                this.health = health;
                this.mana = mana;
                this.armor = armor;
            }
            attack(){
                console.log('파이어볼');
            }
        }
        const x = new Wizard(545, 210, 10);
        console.log(x.health, x.mana, x.armor, x.attack);
         x.attack();

        // 출력
        // 545 210 10
        // 파이어볼
  • 설명
  • 1. js에서 클래스를 작성하라는 것은 객체 단위로 코드 그룹화 및 코드 재사용성을 사용하기 위함.
  • 2. constructor메서드는 class 내에서 객체를 생성하고 초기화하기 위한 특별한 메서드이다.
