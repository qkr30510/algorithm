문제 1: 배열의 내장함수

부분에 배열 내장함수를 이용하여 코드를 입력하고 다음과 같이 출력되게 하세요.

        데이터
        var arr =[200, 100, 300];
        //pass
        console.log(arr);

        출력
        [200,100,10000,300]

답

            var arr = [200, 100, 300];
            //pass
            //console.log(arr);
            var newArr = arr.splice(arr,2)
            var fArr = [...newArr,10000,300]
            console.log(fArr)

해석

  • splice를 이용해서 300을 먼저 지우고 스프레드 연산자를 이용해서 새롭게 생성하여 넣었다.

답지에서의 답

            var arr = [200, 100, 300];
            arr.splice(2, 0, 10000); 
            // splice() 메서드는 배열의 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경합니다.
            console.log(arr);
  • splice() 메서드만 사용해도 되는데 괜히 스프레이드 연산자까지 사용했다.
  • splice() 메서드는 array.splice(배열의 변경을 시작할 인덱스[, 제거할 요소의 수[, 배열에 추가할 요소1[, 배열에 추가할 요소2[, ...]]]])
