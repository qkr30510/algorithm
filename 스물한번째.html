문제1)set은 어떻게 만드나요?

  • 다음 중 set을 만드는 방법으로 올바른 것을 모두 고르시오.
  • 1. var x = {1,2,3,5,6,7}
  • 2. var x = {}
  • 3. var x = new Set('javascript')
  • 4. var x = new Set(range(5))
  • 5. var x = new Set();

정답:3,5

  • 해설
  • set을 무엇을 말하는지 모르겠지만, 함수라고 가정한다면 함수를 만드는건 3,5번이기 때문에 3,5번이라고 말하였다.

문제2) 배수인지 확인하기

  • 다음 중 변수 i가 6의 배수인지 확인하는 방법으로 올바른 것은?
  • 1. i/6 == 0
  • 2. i % 6 == 0
  • 3. i & 6 == 0
  • 4. i | 6 == 0
  • 5. i // 6 == 0

정답:2

  • 해설
  • 배수는 나머지가 0이여야하기때문에 나머지연산자를 사용한 2번이라고 답하였다.
