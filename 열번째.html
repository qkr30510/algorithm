문제 1: 배열의 삭제

다음 배열에서 400, 500을 삭제하는 code를 입력하세요.

var nums =[100, 200, 300, 400, 500]

답

            var num = [100, 200, 300, 400, 500];
            var newNum = num.filter(function(element, index, array){
                return index < 3
                //console.log('element',element);
                //console.log('index',index);
                //console.log('array',array);
            })
            console.log(newNum)

해석

  • filter를 이용하는 문제이다.
  • filter(function(element,index,array){}) 외울것.

답지에서의 답

            var nums = [100, 200, 300, 400, 500];
            nums.pop();
            nums.pop();
            
            console.log(nums);
  • 난 pop()메서드를 몰랐는데, pop메서드를 이용하여 손 쉽게 배열의 마지막을 지우는 방법을 알게 되었다. 그냥 간단하게 pop()을 사용하면 배열의 맨 마지막 요소를 제거해준다.
